# Ta’yaqan

Web-based system for verifying the completeness and authenticity of Qur’anic recitations in recorded audio or video.

## Features
- Upload audio/video or YouTube link
- Convert recitation to text using Whisper ASR
- Compare with authenticated Qur’anic text
- Detect missing, added, or substituted words
- Generate detailed verification report

## Tech Stack
- Python
- Flask
- Whisper
- PostgreSQL
- FFmpeg
